# KittyPaws
Aplicación móvil diseñada para agilizar el proceso de adopción de mascotas, conecta a rescatistas independientes con personas interesadas en brindar un hogar responsable.

Integrantes:
Saavedra Ramírez Vanessa Guadalupe
Cerpa Rodríguez Perla Esmeralda

Tecnologias utilizadas:
Google Antigravity 
React Native y Expo SDK 54.0.8 - se requiere trabajar con la version 54 de Expo debido a que la aplicación movil está desactualizada para 55
Expo Go -  visualizacion movil
Zustand - simula una base de datos de manera online
Expo Image Picker & document picker - para seleccionar imagenes para ávatares de usuario, fotos de mascotas y subir los documentos de verificación


Funcionalidades:
Registro de usuarios adoptantes y dadores
Verificación de identidad (simulación), solicita la carga obligatoria de documentos probatorios de identidad
Catálogo de mascotas disponibles para la adopción
Preguntas, comentarios y reseñas en los perfiles de las mascotas y de los dadores / adoptantes
Cuestionario de postulación para la adopción
Chat entre adoptantes y rescatistas
Seguimiento post-adopción (linea del tiempo)
Sistema de reputación para adoptantes / rescatistas


Instrucciones de instalación o ejecución:
# 1. Clonar el repositorio
git clone https://github.com/VaneGSR/KittyPaws.git
cd "Kitty Paws"

# 2. Instalar dependencias
npm install

# 3. Iniciar servidor de desarrollo
npm start

Conclusión
Este proyecto es una aplicación real de uso cuando se requiere solucionar un problema relativamente complejo, busca dar solución a la necesidad de verificación de perfiles de adoptantes y de rescatistas al dar animales en adopción, lo que reduce la incertidumbre en el proceso, permite dar difusión a las mascotas disponibles y comunicarse de manera transparente con ambas partes.
Es una aplicación sencilla, pero al requerir el uso de perfiles simulados, puede resultar un tanto genérica, hay varios puntos de mejora que pueden abordarse para hacer una aplicación más fluida y completa.
